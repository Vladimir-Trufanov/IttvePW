Быстрый старт

Шаг-1. Включите, указанную ниже строку кода в раздел HTML-страницы со ссылкой на исходный файл tinymce.min.js:

 

<script src="https://cloud.tinymce.com/5/tinymce.min.js?apiKey=your_API_key"></script>

 

В этой строке с вызовом редактора tinymce указывается ключ apiKey, выдаваемый в вашем аккаунте официального сайта TinyMCE.

 

Полный фрагмент HTML-кода предоставляется на шаге 2.

 

Шаг-2. Инициализируйте TinyMCE как часть веб-формы.

После подключения редактора инициализируйте его для любого элемента (или элементов) на веб-странице через селектор CSS.

В указанном ниже примере html-тег

 

<!DOCTYPE html>

<html>

<head>

  

      src='https://cloud.tinymce.com/5/tinymce.min.js?apiKey=your_API_key'>

  

   

      tinymce.init

      ({

         selector: '#mytextarea'

      });

   

</head>

<body>

<h1>TinyMCE Quick Start Guide</h1>

<form method="post">

  

</form>

</body>

</html>

 

 

Шаг-3. Сохраняйте контент через POST-запрос от html-формы.
